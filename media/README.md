# Media folder

Put all site media here:

- **Song:** `pocketful-of-sunshine.mp3` (background music on the valentine page)
- **Videos:** e.g. `goofy.mp4` (polaroid video)
- **Images:** use the filenames referenced in `valentine.html` (e.g. `our-first-trip.jpg`, `sunset-moments.jpg`) or update the HTML to match your files.

Supported: `.mp3`, `.mp4`, `.webm`, `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
